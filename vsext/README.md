# DotArray viewer

## Overview
This is my personal debugging tool. I don't like `0b` command in VS debugger
because it doesn't allow me to change a specific bit which was important for
the original project and I made this extension. 

## How to install DotArray viewer in Visual Studio
1. Install Visual Studio extension and dotNET Desktop development in Visual Studio Installer
1. Open `BinaryVisualizer.sln` in Visual Studio 2026
1. Compile the solution
1. Install `bin\vsix\Release\x64\BinaryVisualizer.vsix`
